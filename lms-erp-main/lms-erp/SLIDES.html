Key Features

🎓 Course Management

Structured courses with sections, video lessons, downloadable documents, and multiple-choice quizzes with pass/fail scoring.

📊 Progress Tracking

Per-lesson completion tracking, quiz scores, enrollment status, and overall course progress percentage for every employee.

🏆 Auto Certificates

Certificates are automatically generated when an employee completes a course. Each has a unique number and is publicly verifiable.

👥 Bulk User Import

Upload a CSV file with 300+ employees — name, email, employee ID, department, role. All accounts created instantly with a default password.

Bulk Employee Import

📄 CSV Format

Jane Doe,jane@arohak.com,EMP002,IT,instructor
...300 more rows

⚡ How It Works

  • Admin clicks "Import CSV"
  • Upload the prepared CSV file
  • System creates all accounts
  • Duplicate emails auto-skipped
  • Default password: Welcome@123

📊 Import Result

Shows total rows, created count, skipped count, and notes for any issues — all in one summary screen.
